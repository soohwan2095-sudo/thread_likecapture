Thread LikeCapture User Guide

This document matches the current single-screen batch workflow: choose a source folder, choose a Gemini preset, define categories, optionally enter a session-only Gemini API key, then press Start.

The Korean PDF guide is the primary manual: docs\사용방법_안내.pdf

Current Workflow

  1. Run startup.bat.
  2. Put source files into data\source-files.
  3. Choose a Gemini preset on the main screen.
  4. Enter categories such as 경제, 바이브코딩, 기타.
  5. Enter a Gemini API key only if you want live analysis.
  6. Press 시작.
  7. Review output PDFs in the archive category folders.

Folder Layout

D:\HPCodes\thread_likecapture
├─ data
│  ├─ source-files
│  └─ archive
│     ├─ 경제
│     ├─ 바이브코딩
│     ├─ 기타
│     └─ meta
└─ docs

Security Notes

  • The Gemini API key is session-only in the UI.
  • The app does not automate browser login or control your Google account.
  • The app only sends the files you selected for Gemini analysis.
