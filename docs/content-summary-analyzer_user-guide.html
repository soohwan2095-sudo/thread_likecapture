글 요약 분석기 User Guide

원본 폴더에 PDF, PNG, JPG 파일을 넣고 Gemini 모델과 카테고리를 선택한 뒤 시작 버튼을 누르면 요약과 분석 결과를 PDF로 정리합니다.

한국어 PDF 안내서는 docs\사용방법_안내.pdf입니다.

기본 사용 흐름

  1. startup.bat을 실행합니다.
  2. 원본 파일을 data\source-files에 넣습니다.
  3. 메인 화면에서 Gemini 모델을 선택합니다.
  4. 카테고리를 쉼표 또는 줄바꿈으로 입력합니다.
  5. 실제 분석이 필요하면 Gemini API 키를 입력합니다.
  6. 시작 버튼을 누릅니다.
  7. data\archive 아래 카테고리 폴더에서 결과 PDF를 확인합니다.

폴더 구조

D:\HPCodes\content_summary_analyzer
├─ data
│  ├─ source-files
│  └─ archive
│     ├─ 경제
│     ├─ 바이브코딩
│     ├─ 기타
│     └─ meta
└─ docs

보안 메모

  • Gemini API 키는 화면 세션에서만 사용합니다.
  • 브라우저 로그인 자동화나 구글 계정 제어를 하지 않습니다.
  • Gemini 분석 요청에는 사용자가 지정한 원본 파일만 보냅니다.
