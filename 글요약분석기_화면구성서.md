# 글 요약 분석기 화면구성서

## 1. 문서 목적

현재 구현된 단일 화면 UI 구성을 정리한다.

## 2. 전체 구조

```text
상단 Hero
좌측 제어 패널
우측 상태 / 파일 / 결과 패널
```

## 3. 상단 Hero

- 프로그램 이름: `글 요약 분석기`
- 짧은 설명 문구
- 현재 선택된 Gemini 모델 표시

## 4. 좌측 제어 패널

- 원본 폴더 카드: 원본 파일 경로, 결과 저장 경로, 폴더 열기, 경로 적용
- AI 모델 카드: Gemini 모델 프리셋 선택
- 카테고리 생성 카드: 카테고리 입력, 태그 미리보기, 폴더 생성
- 보안 설정 카드: 세션용 Gemini API Key 입력
- Auto Run 카드: 자동 실행 토글, 감시 주기 입력
- Batch Run 카드: 시작 버튼

## 5. 우측 결과 패널

- 현재 상태 카드: 상태 메시지, 새로고침, 신규/건너뜀/실패 요약
- 원본 폴더 파일 목록: 파일명, 상태, 카테고리, 원본 열기, 결과 열기
- 최근 결과: 최근 처리 결과 카드 목록
- 선택한 결과: 카테고리, 한 줄 요약, 상세 해설, 핵심 포인트, 인사이트, 원본/결과 PDF 열기

## 6. 사용자 흐름

1. 원본 폴더와 결과 폴더를 확인한다.
2. Gemini 모델을 선택한다.
3. 카테고리를 입력한다.
4. API 키를 입력하거나 데모 모드를 사용한다.
5. `시작` 또는 `Auto Run`으로 실행한다.
6. 상태 카드에서 진행 결과를 확인한다.
7. 최근 결과에서 상세 결과를 연다.
